A cookie is a bit of data stored by the browser and sent to the server with every request.
So, in every request browser remembers that this cookie belongs to this user.



A session is a collection of data stored on the server and associated with a given user (usually via a cookie containing an id code)
A session is stored on the server while a cookie is stored on the browser.



passport is for the authentication part
passport-local is used for the local login.
